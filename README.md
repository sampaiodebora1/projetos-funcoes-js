# Project: JavaScript Functions

This project was developed to present fundamental concepts of functions in JavaScript using Node.js as the execution environment.

## Concepts explored

- Regular function
- Anonymous function
- Arrow function
- Parameters and return values
- Execution using Node.js

## Technologies used

- JavaScript
- Node.js
- VS Code

## How to run

In the terminal, run:

node index.js

## Objective

The goal of this project is to understand how functions work in JavaScript and practice different ways of defining and executing them.
